Arduino Electrónica

- Resumen -

Mediante la realización de estas prácticas se logrará un mejor entendimiento del funcionamiento de electrónica, arduino y la programación de este.

- Introducción -

Para estas prácticas no es necesario tener conocimientos previos, de momento solo se requerirá seguir los pasos indicados.

- Materiales -

Arduino UNO, software Arduino y archivos descargables de la práctica.

Construcción

Por el momento solo será necesario conectar Arudino mediante el cable USB a la comptadora.

- Desarrollo -

Como primer paso abrimos el código arduino, antes de compilarlo y/o iniciar el código debemos especificar el tipo de arduino que vamos a utilizar, en nuestro caso Arduino UNO. Una vez seleccionado damos click en "ok". Observamos que en las líneas de código 3 y 8 tenemos "void setup" y "void loop" respectivamente, el primero sirve para asignar el pin con el que trabajaremos, el segundo es donde le daremos las instrucciones que queremos que realice con la característica de que ejecutará las órdenes de forma infinita. El código llamará al led "L" para que se mantenga encendido.

Simulación

Para iniciar el código primero compilamos/verificamos el código esté bien, para ello damos click en la palomita que se encuentra en la parte superior izquierda, si todo es correcto nos aparecerá "done compiling", acto seguido damos click en el botón "upload" ubicado junto al botón que usamos para compilar el código, con upload hacemos que el código lo ejecute arduino.

- Resultados -

Podemos observar que el led junto a "L" se mantiene encendido, como lo indican las instrucciones dadas.

- Conclusiones -

Aprendimos a llamar a un pin en específico y asignarle órdenes a seguir.
