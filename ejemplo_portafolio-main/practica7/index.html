Arduino Electrónica
<

- Resumen -

Mediante la realización de estas prácticas se logrará un mejor entendimiento del funcionamiento de electrónica, arduino y la programación de este.

- Introducción -

Para estas prácticas no es necesario tener conocimientos previos, de momento solo se requerirá seguir los pasos indicados.

- Materiales -

Arduino UNO, software Arduino, 2 led rojo, 1 protoboard, 6 cables macho-macho, 2 resistencia de 100 ohm's y archivos descargables de la práctica.

Construcción

Conectamos arduino a nuestra computadora y nos aseguramos de conectarla con el protoboard siguiendo la imagen de referencia, recuerda colocar de manera correcta el led rojo.

- Desarrollo -

Abrimos arduino y lo configuramos como en las anteriores prácticas. En este código podremos encender y apagar 2 leds al mismo tiempo con valores que podremos variar.

Simulación

Primero compilamos/verificamos que el código no contenga errores, preparamos el protoboard como se ve en la imagen y enviamos el código para comenzar a simular.

- Resultados -

Observamos que los led rojo se aprecian parpadeando, están prendiendo y apagándose como lo indican las instrucciones dadas, puedes jugar con estos valores para apreciar como afecta en el comportamiento de arduino y protoboard.

- Conclusiones -

Podemos utilizar una gran cantidad de elementos electrónicos a la vez y configurarlos.
